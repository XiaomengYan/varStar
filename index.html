Varstar

Period Estimation for Quasi-Periodical Mira Variable Stars

View project on GitHub

R Package

VarStar is an R package to fit Mira light curves with Gaussian process model. Two versions of the model are provided, one sophisticated version for densely sampled Mira light curve, and one simple sinusoid version for sparsely sampled light curve.

Installation

Prerequisite R packages are Rcpp, RcppArmadillo. Use the following code to install from Github.

library(devtools)
install_github("shiyuanhe/varStar/varStar")
